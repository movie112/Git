# VSCODE 단축키 정리

- `alt + ⬆⬇` : 이동
- `ctrl + alt + ⬆⬇` : 세로 선택
- 'shift + alt + ⬆⬇` : 라인 복사
- `ctrl + x` : 라인 삭제
- `ctrl +g` : 줄 이동
- `alt + ⬅➡` : 이전 작업
- `ctrl + h + f` : 정렬
- `shift + ctrl + i` : 블럭커서 맨 뒤
- `ctrl + (shift) + Enter` : 빈 줄 생성
- `ctrl + click`
